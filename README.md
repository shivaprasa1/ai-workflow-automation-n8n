# AI Workflow Automation System (n8n)

## 🚀 Overview
This project is an AI-powered workflow automation system built using n8n.

## ⚙️ Features
- Webhook-based trigger
- Conditional logic using IF node
- AI integration using OpenRouter API
- Automated email responses using Gmail

## 🔄 Workflow
Webhook → IF → AI → Gmail

## 🧠 Technologies
- n8n
- REST APIs
- Webhooks
- OpenRouter (LLM API)
- Gmail API
